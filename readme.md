# generator-js

> A Yeoman generator for scaffolding my JavaScript modules.

Heavily inspired by [`generator-nm`](https://github.com/sindresorhus/generator-nm)!

## Install

```sh
$ npm i -g yo @tomer/generator-js
```

## Usage

With [`yo`](https://github.com/yeoman/yo):

```sh
$ yo @tomer/js
```
